# InstiApp
The IIT Bhubaneswar App

## Current Features
- Campus Map
  - Hybrid view with constrained zoom and location
  - Gestures enabled
- Transport Schedule
- Timetables
- Academic Calendar
- Public Holidays
- Regulations
- ERP integration
- Students' Gymkhana section

## Features Coming Soon
- Search and navigation in map
- News Feed
- Dual color scheme for night mode
